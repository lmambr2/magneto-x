# [¶](#h-1-unboxing) 1. Unboxing

 For earlier versions, please refer to this installation guide: [https://wiki.peopoly.net/en/magneto/magneto-x/enclosure-setup-guide](/en/magneto/magneto-x/enclosure-setup-guide)

 After receiving the enclosure kit, open the package.

 The package contains acrylic, aluminum profiles, screws, hinges and other items needed for enclosure the box.

 Each small package has a corresponding label. Please refer to these labels and assemble according to the subsequent instructions.

# [¶](#h-2-introduction-enclosure-main-framework) 2. Introduction enclosure main framework

 Before assembling, we need to name the various components of the enclosure kit. Please note that the names of the win the bidding annotations shown below are all in the form of labels attached to the corresponding items.

## [¶](#h-21-main-body) 2.1 Main Body

 If you have purchased a Jetstream fan, please install acrylic Bootom-Side-B1 and Bootom-Side-B2 with holes. If you have not purchased a Jetstream fan, please install acrylic Bootom-Side-B3 and Bootom-Side-B4 without holes. Bootom-Side-B3 and Bootom-Side-B4 do not have fan holes.

## [¶](#h-22-top-cover) 2.2 Top Cover

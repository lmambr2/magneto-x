Magneto X booting issue | Peopoly Wiki - - - - - - - -

 This document provides temporary solutions when the Linux system keeps restarting or the system fails to start.

 Please try to find the location shown in the figure below of Magneto X, which is next to the printer network port. There is a Type C interface. Please provide an additional 5V(1.5A+) power supply to this. See if the system can start normally.

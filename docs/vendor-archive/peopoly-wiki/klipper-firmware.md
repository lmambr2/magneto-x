# [¶](#magneto-x-klipper-firmware) Magneto X Klipper Firmware

 For the official Magneto X firmware mirror download, please follow the [Klipper firmware installation guide here](/en/magneto/magneto-x/update-tf-image).

 [Peopoly GitHub Repository](https://github.com/mypeopoly/Klipper)

 Klipper is a 3D printer firmware that enhances printing performance by combining the power of general-purpose computing with microcontrollers. It is known for its high precision and speed capabilities. It gained popularity first through Voron Design printers and then with printers OEM adopting it for Corexy, i3, and other FDM 3D printers.

 Magneto X utilizes a customized version of Klipper (V0.11), with the code detailing specific customizations available on Peopoly's GitHub—the customization primary focus on communication protocols with loadcell sensor and linear motor driver. Peopoly open sources these codes to uphold the open source licensing agreement and contribute back to the community that made Magneto X possible. Peopoly is also a proud sponsor of the Klipper project.

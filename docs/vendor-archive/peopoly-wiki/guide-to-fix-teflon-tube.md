# [¶](#guide-to-attaching-a-teflon-tube-to-the-y-axis-motor) Guide to Attaching a Teflon Tube to the Y-axis Motor

 This document guides you on how to attach a Teflon tube to the Y-axis motor, securing the Teflon tube clamp to the motor.

## [¶](#h-1-download-and-print-the-model) 1. Download and Print the Model

 Download the STEP file from this link:

[https://drive.google.com/file/d/1FipdZ-E0z5yL442zqRPtSkqzNDwr5kKJ/view?usp=sharing](https://drive.google.com/file/d/1FipdZ-E0z5yL442zqRPtSkqzNDwr5kKJ/view?usp=sharing)

 Import it into Orcaslicer.

 We recommend using the Lancer PETG-C that comes with the Magneto X for printing. Please use the parameters shown in the figure below for printing.

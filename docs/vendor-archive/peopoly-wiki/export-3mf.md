# [¶](#guide-to-exporting-3mf-files) Guide to Exporting 3MF Files

## [¶](#h-1why-export-as-3mf-format) 1.	Why Export as 3MF Format

 A 3MF file includes not only the model data but also slicing information, selected printer type, nozzle diameter, layer height, printing speed, and all other parameters. This comprehensive information is crucial for the accurate reproduction of the model.

 When analyzing the causes of poor printing results, the 3MF file provides detailed data that assists us in making more refined optimizations.

## [¶](#h-2-how-to-export-a-3mf-file) 2. How to Export a 3MF File

 After importing the model into the slicing software, select the appropriate printing parameters, such as material type, layer height, printing speed, etc., and then export it as a 3MF file.

# [¶](#feature-and-specification-overview) Feature and Specification Overview

 Brand: Peopoly

## [¶](#general-information) General Information

 ●	Product Name: Magneto X

●	Product Type: Desktop FFF 3D Printer

●	Launch Date: September 26th, 2023

●	Retail Price: US$1999

●	Warranty Period: 1 Year on Parts

●	Warranty Scope: Worldwide

## [¶](#mechanical-system) Mechanical System

 ●	Motion System: MagXY

●	Type: Magnetic levitation linear motor motion system for X and Y axes

●	Movement Precision: 3um

●	Error Correction: Full Closed Loop

●	Max Print Speed: 800mm/s at 0.2mm layer height (0.4mm nozzle)

●	Max Travel Speed: 1500mm/s

●	Maximum Acceleration: 22000mm/s²

●	Safety Feature: Auto Disengagement of the motor when the extruder runs into an object to prevent extensive damage

●	Z-Axis: 4 Independently driven linear rails with lead screws

 Size:

●	Print Volume: 400 x 300 x 300 mm (X/Y/Z)

●	Build Platform: 400 x 300 mm

●	Printer Size: 55 x 61 x 70 cm

 Build Plate:

●	Build Platform Material: Aluminum

●	Build Surface: PEI Magnetic Flexible Build Surface

●	Heating Power: 1000W AC

●	Leveling: 48 Mesh Points Auto-Leveling via Load Cell Sensor

 Enclosure:

●	Side Enclosure: Optional semi-clear smoky color acrylic panels on four sides.

●	Top Enclosure: Optional top cover

## [¶](#extrusion-system) Extrusion System

 ●	Extruder Name: Lancer

●	Maximum Extrusion Force: 90 Newton

●	Maximum Filament Speed: 30mm/s

●	Gear Ratio: 7.5:1

●	Maximum Flow Rate: 60 mm³/s (using included melt zone)

●	Leveling Sensor: Load Cell

●	Filament Runout Sensor: Yes, with filament movement control

●	Data Connection: USB

●	Hot End Maximum Temperature: 300C

●	Nozzle Connection: RepRap M6 / E3D V6

●	Nozzle Material: Copper, Hardened Steel

●	Nozzle Coating: Teflon

●	Melt Zone: Swappable

●	Cooling Fans: 3

- 4015 13000 RPM brushless turbo fan x2 for parts

- 3010 8000 RPM ball bearing fan x1 for heat sink

 ●	Supported materials: PLA, PETG, ABS, ASA, PA6, PA12, PA-CF, PC-GF, TPU, and more

## [¶](#electrical-system) Electrical System

 Power:

●	Supply Input: 110V / 220V

●	Max Power: 1950W, 48V 600W + 24V 350W + 1000W AC

 Control and Interface:

●	System Board: Orange Pi Zero 2 H616

●	Control board: BTT Octopus Pro 1.1 H712

●	Onboard Storage: 32GB TF Card

●	WiFi: 802.11 N/AC

●	Ethernet: 1Gb

●	Screen: 7.0-inch IPS touchscreen

●	Firmware: Klipper

●	Slicer Compatibility: Orca Slicer, PrusaSlicer, Cura

## [¶](#auxiliary-systems) Auxiliary Systems

 ●	Camera: 1080P 30FPS

●	2nd Display: Holographic Cube for displaying Gcode’s 3D shape

●	Sensor: Loadcell Sensor, Filament Run Out Sensor
